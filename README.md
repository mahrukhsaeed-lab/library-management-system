# Library Management System

A simple Library Management System developed in Python.

## Features

- Add new books
- View all books
- Search for books
- Issue books
- Return books
- Delete books
- Easy-to-use menu interface

## Technologies Used

- Python 3

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/mahrukhsaeed-lab/library-management-system.git
   ```

2. Navigate to the project folder:
   ```bash
   cd library-management-system
   ```

3. Run the program:
   ```bash
   python main.py
   ```
